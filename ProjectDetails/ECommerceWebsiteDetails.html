Project Overview

This is a basic static e-commerce website created using only HTML, CSS, and JavaScript. It showcases products in a clean and responsive layout. Users can browse product and view product details. All functionality is handled on the front-end with no back-end integration, making it perfect for learning static site development and basic DOM manipulation.

Key Features

  • Static product catalog
  • Product detail pages
  • Responsive design for mobile and desktop
