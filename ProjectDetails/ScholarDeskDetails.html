ScholarDesk Web Application

A clean and easy-to-use university management system for students, teachers, and admins.

PHP HTML CSS JavaScript phpMyAdmin

Project Screenshots

Project Overview

Teachers, admins, and students can register and log in. Students log in using their matriculation number and password, while teachers and admins use their email and password. I created ScholarDesk because the university management system provided by my school was confusing and difficult to use. I wanted to build a simpler, more user-friendly system. Unlike the full system at my university (which includes hostel selection and school fee payments), ScholarDesk focuses only on helping students manage their courses and allowing teachers to select which courses they want to teach, manage students, and track attendance. With ScholarDesk, admins can add and edit departments, create and delete courses, remove students from courses, update grades, and manage teachers in courses. Teachers can select the courses they want to teach, view enrolled students, manage grades and attendance, and edit their profiles. Students can enroll in courses, edit their profiles, and view their registered courses.

Key Features

  • Built with PHP, HTML, CSS, JavaScript, and phpMyAdmin
  • Students can register, log in, enroll in courses, and edit profiles
  • Teachers can manage courses, grades, and attendance
  • Admins can manage departments, courses, and user roles
  • Simple matriculation number and password login for students
  • Clean, intuitive dashboard for students, teachers, and admins
  • Fully responsive design
