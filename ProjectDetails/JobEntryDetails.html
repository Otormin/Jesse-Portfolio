JobEntry Web Application

An online platform connecting recruiters and job seekers for smooth hiring.

Project Screenshots

Project Overview

I created JobEntry to help tackle the high unemployment rate in Nigeria by providing a platform that directly connects recruiters and talents. Recruiters can post, edit, and delete job listings, which can be full-time or part-time. Talents can browse available jobs and apply by sending cover letters. Each user type has a dashboard: recruiters can view all applicants for their jobs and select the best candidates, while talents can track their applications, see which ones have been accepted, and withdraw applications if they wish. Talents can also compose and share posts for recruiters and other talents to see.

Key Features

  • Built with PHP, HTML, CSS, and JavaScript
  • Recruiters and talents can register and log in
  • Recruiters can post, edit, and delete job listings
  • Jobs can be marked as full-time or part-time
  • Talents can browse jobs and apply with cover letters
  • Recruiters have a dashboard to manage applications and select candidates
  • Talents have a dashboard to track applications and job statuses
  • Talents can compose and share posts for recruiters and other talents to see
  • Talents can withdraw job applications if needed
  • Database managed using phpMyAdmin
  • Responsive design for smooth user experience
