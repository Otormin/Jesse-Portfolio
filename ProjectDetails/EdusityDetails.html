Project Overview

Edusity is a modern university landing page designed to showcase academic programs, campus experiences, and student success stories. Built using React and Vite, it features smooth page transitions, an interactive navigation bar, animated hero section, testimonial carousel, video modal for campus tours, and a responsive photo gallery with lightbox functionality.

Key Features

  • Fully responsive design compatible with all devices
  • Smooth scrolling navigation with active link indicators
  • Animated Hero Section with Call-to-Action Buttons
  • Program Showcase with Hover Effects
  • Video Modal Popup for Campus Tour
  • Dynamic Gallery with Lightbox Preview Mode
  • Testimonials Carousel Slider
  • Functional Contact Form with Validation
  • Component-based architecture with scoped CSS Modules
  • Performance optimized with lazy loading
