Project Screenshots

Project Overview

Moviez is a real-time movie search and discovery platform powered by TMDB API and Appwrite. It allows users to search thousands of movies with live suggestions, explore detailed information such as ratings, languages, and release dates, and view trending movies intelligently ranked based on user search activity.

Key Features

  • Real-time debounced movie search for optimized API usage
  • Trending movies automatically tracked using Appwrite database
  • High-quality movie cards with hover effects
  • Detailed movie pages with ratings, language, and release year
  • Responsive and mobile-friendly layout
  • Secure environment variable configuration
