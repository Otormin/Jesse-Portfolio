FarmConnect Web Application

A web platform connecting farmers, workers, and consumers to buy farm products directly and apply for farm jobs.

PHP HTML CSS JavaScript phpMyAdmin

Project Screenshots

Project Overview

I created FarmConnect to tackle rising food prices in Nigeria caused by middlemen. This web application connects farmers directly to consumers, helping lower product costs. It also addresses high unemployment by letting farmers post farm jobs that workers can apply for. Farmers can manage products and job postings, while consumers can buy produce in different categories and add items to a cart. Workers can browse jobs, apply with a cover letter, and track their application status.

Key Features

  • Built with PHP, HTML, CSS, and JavaScript
  • Farmers, workers, and consumers can register and log in
  • Farmers can post, edit, and delete farm products
  • Products categorized (fruits, vegetables, meat, fish, grains, legumes, eggs, dairy, others)
  • Consumers can select products, set quantity, and add to cart
  • Farmers can post, edit, and delete jobs (farming-machinery or farming-tools)
  • Workers can browse jobs, apply with cover letters, and manage applications
  • Farmers can view and accept or reject job applications
  • Workers can track application status and withdraw if needed
  • phpMyAdmin used for database management
