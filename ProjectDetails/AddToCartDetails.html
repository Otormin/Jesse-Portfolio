Project Overview

This is a simple grocery list web app built with HTML, CSS, JavaScript, and Firebase. Users can add grocery items to a database, retrieve them, and delete them when needed. The app uses Firebase to handle real-time data storage and retrieval. This project helped me learn how to connect a front-end web app to a cloud database using Firebase.

Key Features

  • Add grocery items to a Firebase database
  • Retrieve and display grocery list from Firebase
  • Delete items from the database
  • Uses Firebase Realtime Database for cloud storage
  • Simple, responsive design
  • Clean user interface for easy use
