Project Screenshots

Project Overview

RunGroup is a web application built with ASP.NET MVC, JWT, and SQL Server. Users can discover and join different running clubs, select the runs they want to participate in, and manage their activities easily. The app provides features for users to create, edit, or delete their runs and clubs. It uses Cloudinary for image storage, IPinfo for location data, and Bootstrap for a clean, responsive design.

Key Features

  • Join and manage running clubs
  • Select and sign up for runs
  • Create, edit, and delete runs
  • Create, edit, and delete clubs
  • Store images with Cloudinary
  • Use IPinfo for location data
  • Built with ASP.NET MVC and secured with JWT
  • Responsive design with Bootstrap and HTML
