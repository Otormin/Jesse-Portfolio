Project Overview

This is a simple chatbot built with HTML, CSS, and JavaScript that connects to the Gemini API. Users can type questions or upload images, and the chatbot responds in real time with text or relevant insights based on the uploaded image. This project helped me learn how to handle file uploads, work with APIs, and build dynamic chat interactions in a clean web interface.

Key Features

  • Built with HTML, CSS, and JavaScript
  • Uses the Gemini API for AI responses
  • Supports uploading images for analysis
  • Handles text and image inputs
  • Real-time chat interaction
  • Responsive design for desktop and mobile
