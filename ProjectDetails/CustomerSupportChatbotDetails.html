Restaurant Customer Support Chatbot

A conversational AI chatbot built with Google Dialogflow for restaurant customer support, handling table bookings, food ordering, and general inquiries.

Project Screenshots

Project Overview

This chatbot serves as a virtual assistant for restaurant customers, providing 24/7 support for common tasks like table reservations, menu inquiries, and food ordering.

Key Features

  • Table Booking System with automatic date recognition
  • Friendly Welcome & Greeting messages for new customers
  • Menu Inquiry: Check availability of menu items
  • Food Ordering: Place orders directly via chatbot
  • Smart Fallback: Handles unrecognized queries
  • Natural Language Understanding with Dialogflow
  • Custom entities and system entities for data extraction
